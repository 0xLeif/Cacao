# Cacao
Pure Swift Cross-platform UIKit (Cocoa Touch) implementation (Supports Linux)

## Installation

### OS X
`brew install cairo sdl2`

### Ubuntu
`sudo apt-get install libcairo-dev libsdl2-dev`
