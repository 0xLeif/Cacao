Pure Swift Cross-platform UIKit (Cocoa Touch) implementation (Supports Linux)

## Dependencies

### OS X
`brew install cairo sdl2 lcms2`

### Ubuntu
`sudo apt-get install libcairo-dev libsdl2-dev liblcms2-dev`
